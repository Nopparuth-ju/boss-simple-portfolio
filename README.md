# Nopparuth Boss - Data Analyst Portfolio

A modern, minimalist portfolio website designed for a Data Analyst. This project is hosted on **GitHub Pages** and features a clean dark theme using the JetBrains Mono font.

## 🚀 Live Demo
View the live portfolio at: [https://nopparuth-ju.github.io/boss-simple-portfolio/](https://nopparuth-ju.github.io/boss-simple-portfolio/)

## ✨ Features
- **Minimalist Design**: Clean UI focused on readability and professional identity.
- **Dark Mode Support**: Built-in dark theme with toggle functionality.
- **Responsive Layout**: Designed to look great on various screen sizes.
- **Typography**: Uses **JetBrains Mono** for a technical, data-driven aesthetic.

## 🛠️ Built With
- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom styling with a modern dark theme.
- **JavaScript**: Interactive features like theme toggling.
- **Google Fonts**: JetBrains Mono for typography.

## 📁 Project Structure
- `docs/index.html`: Main structure of the portfolio.
- `docs/style.css`: Custom styles and theme definitions.
- `docs/script.js`: Interactive logic (Theme toggling).
- `portfolio-website.md`: Project planning and roadmap.

## 🚀 Getting Started
To view this portfolio locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/Nopparuth-ju/boss-simple-portfolio.git
   ```
2. Open `docs/index.html` in your favorite web browser.
